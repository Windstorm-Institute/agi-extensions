That repository holds the current manuscript (v1.5), the internal adversarial review, the Grand Slam supplementary materials, the complete experiment tree, the published article, and the Paper 7.1 tracking issue.

## Why the rename?

This work was originally drafted under the working title *AGI Extensions*. When the v1.4 manuscript was completed, the title became *The Throughput Basin Origin* to match the rest of the Windstorm series naming convention (Fons Constraint, Receiver-Limited Floor, Throughput Basin, Serial Decoding Basin τ, Dissipative Decoder, Inherited Constraint). This redirect repository is preserved so older links continue to resolve.
